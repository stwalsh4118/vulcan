## Requirements

1. Initialize `go.mod` with module path `github.com/seantiz/vulcan` (or appropriate path) and Go 1.23+.
2. Create directory structure:
   - `cmd/vulcan/` — main entry point
   - `internal/api/` — HTTP handlers and router
   - `internal/store/` — persistence layer
   - `internal/config/` — configuration loading
   - `internal/model/` — domain types
3. Create `cmd/vulcan/main.go` that compiles (`go build ./cmd/vulcan`) and prints a startup message.
4. Add a `.gitkeep` in empty directories or a brief `doc.go` with package comment.
5. `go build ./cmd/vulcan` succeeds without errors.
